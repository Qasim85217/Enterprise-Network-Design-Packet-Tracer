# **Enterprise Network Design using Cisco Packet Tracer**

#### Project Description

##### 

##### This project demonstrates the design and implementation of a secure enterprise network using Cisco Packet Tracer. The network is segmented into multiple VLANs to improve security, traffic management, and network organization.

##### 

#### Features

* ##### VLAN Segmentation
* ##### Router-on-a-Stick Inter-VLAN Routing
* ##### Centralized DHCP Server
* ##### DHCP Relay (IP Helper Address)
* ##### Wireless Network Integration
* ##### Dedicated Server VLAN
* ##### ACL-Based Department Isolation
* ##### Physical and Logical Network Design



### **VLAN Structure**



##### | VLAN | Department | Network         |

##### | ---- | ---------- | --------------- |

##### | 10   | DEPT\_A     | 192.168.10.0/24 |

##### | 20   | DEPT\_B     | 192.168.20.0/24 |

##### | 30   | SECRET     | 192.168.30.0/24 |

##### | 40   | SERVERS    | 192.168.40.0/24 |

##### | 50   | WIFI       | 192.168.50.0/24 |

# 

#### Technologies Used

* ##### Cisco Packet Tracer
* ##### VLANs
* ##### 802.1Q Trunking
* ##### Router-on-a-Stick
* ##### DHCP
* ##### DHCP Relay
* ##### Access Control Lists (ACLs)
* ##### Wireless Networking
* ##### Security Implementation

##### 

##### An Extended Access Control List (ACL) was implemented to isolate the Secret Department (VLAN 30) from other network segments while maintaining DHCP functionality.
